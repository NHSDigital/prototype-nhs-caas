Raising a request

Only a line manager or Service owner can raise a request through ServiceNow.

They can raise a request to:

  • add a new user (the user will need to have Clinical Document Architecture (CDA) and Care ID approval first)
  • select 'something is broken' on the homepage
  • remove a user
  • request an audit report (only service owners can do this)
  • obtain patient cohort information prior to September 2023

How to raise a request

  1. log in to ServiceNow
  2. select 'request something' on the homepage
  3. on the Service catalogue page select 'general request'

Before you submit your report

You need to make sure that the request is sent to the correct team.

To do this, you need to enter the following information in the corresponding fields:

  • Which department you want to report the issue to. Select 'live services' in the drop-down menu
  • Service option. Type CaaS into the field and choose Cohorting as a Service
  • Service offering. Choose Find out why a patient is in a cohort in the drop-down menu

What happens after you submit your report

Once you submit the form, you will receive an email with a ticket number. You will also be sent an email once your problem has been resolved or if any further information is needed.

Reviewing or updating a ticket

To review or update a ticket you need to:

  1. log in to ServiceNow
  2. navigate to top right of the homepage
  3. select ‘my tickets’
  4. select the ‘requested items’ tab to see your open or closed tickets
