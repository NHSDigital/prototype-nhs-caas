Cohort Enquirer

Cohort Enquirer is a self-service tool that allows you to find out if a patient is part of a cohort.

What you need

To search for a patient's cohort information, you will need both their:

  • NHS number
  • date of birth

Before you start

To access Cohort Enquirer, you will need to log in using CIS2 (Care Identity Service 2) authentication.

Contact your line manager if you do not have a CIS2 account.
