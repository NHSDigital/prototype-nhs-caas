Find out why a patient is in a cohort

This tool provides an understanding into why a patient has been included in a cohort. It gives details about the conditions that made them eligible and the data sets and code lists used.

Which patient details you need

To search for a patient's cohort information, you will need both the patient's NHS number and date of birth

Information:

We can only provide patient information processed on or after September 2023.

If you need information from before September 2023, create a request through ServiceNow.

How to access

To access the tool, you must log in through the Care Identity Service (CIS2).
