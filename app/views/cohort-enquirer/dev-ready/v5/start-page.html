Find out why a patient is in a cohort

This tool explains why a patient is included in a cohort. It shows the conditions that made them eligible and the data sets and codes used.

Which patient details you need

To search for a patient's cohort information, you will need both the patient's NHS number and date of birth.

Information:

This tool provides only the latest patient cohort information. For older information, create a request through ServiceNow.

Before you start

To access the tool, you must log in through the Care Identity Service (CIS2).

You will need to make sure JavaScript is enabled in your browser.
