Request a change to an NHS cohort

Use this service if you work on healthcare programmes and need to make simple changes to an NHS cohort.

You can use this service to:

  • view a cohort
  • create a copy of an existing cohort
  • submit your cohort for review

Before you start

You'll need to:

  • know the details of the cohort you want to view or copy
  • confirm you and your team are responsible for any changes made
  • accept any clinical risks
