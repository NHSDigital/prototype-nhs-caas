Cohorting as a Service (CaaS)

Cohorting as a Service securely identifies groups of people with shared characteristics from national health data. The results enable tailored direct care to be delivered to improve health and social care outcomes.

View the libraries that the service has created below.

  • Cohort library

    The cohort library displays the existing cohorts created by Cohorting as a Service.

  • Condition group library

    The conidtion group library displays information on individual condition groups.

  • Data integrations library

    The data integrations library displays the data sources used in the cohorts.
