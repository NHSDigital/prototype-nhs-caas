Accessibility statement for Cohorting as a Service (CaaS)

The CaaS service is run by NHS England. We want everyone to be able to use it.

We're committed to making this service accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This service is compliant with the Web Content Accessibility Guidelines version 2.1 AA standard.

We have self-assessed the service for accessibility against those guidelines using assistive tools, accessibility assessment tools and accessibility testing.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible the service is

When using the service, you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • navigate the service using just a keyboard
  • navigate the service using speech recognition software
  • listen to the service using a screen reader (including the most recent versions of NVDA and VoiceOver)
  • access the service using a supported browser on a modern personal computer or device

On sections that do not provide clinical information, we’ve also made the text as simple as possible to understand.

We cannot guarantee the accessibility of links outside of the CaaS service that we do not own or manage.

To call us using the NGT service, dial 18001 followed by 0300 303 5678.

Reporting accessibility problems with the service

We’re always looking to improve the accessibility of this service.

If you find any problems not listed on this page or think we’re not meeting accessibility requirements, email england.web@england.nhs.uk with details of the problem you have encountered.

Enforcement procedure

The Equality and Human Rights Commission is responsible for enforcing the accessibility regulations.

If you're not happy with how we respond to your complaint, contact the Equality Advisory and Support Service.
